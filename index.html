Vital Vis

Two methods for visualizing patient data are offered: Stacks and Overlay. The Stacks technique allows the user to display all vitals and events for a single file at a time using multiple time series line charts. The Overlay technique allows the user to select a drug and vital to show all patient data overlaid together on single plots, one for each dosage of the drug. Choose a button below to proceed.
