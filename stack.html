Stack Charts

Use the "Open CSV File(s)" button above to read local files into this web page for analysis. The files must be stored in the ORNL CSV format with column headers. Once read, the files can be selected from the middle select component. The "Redraw" button can be used to redraw the charts if the window is resized. Chart settings can be modified using the "Settings" dialog.
